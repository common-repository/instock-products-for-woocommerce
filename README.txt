== Description ==
Instock Products Woocommerce Plugin Help you to show your WooCommerce Store Products Count and Show for "In Store" "Out of Stock" "Sold". So you will happy to use this plugin. is very easy to use, just install and activate plugin, it will work automatically.

* Stock Products Count Show
* In-Stock Products
* Out--of-stock Products
* Sold Item Show

Yes. it will work with any WordPress And WooCommerce Theme


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Just install and active it will work automaticall.
4. Notice: you should click and fill your stock product in edit product.
